# NewsMouse

Welcome to News Mouse! This application utilizes the NewsData.io API to collect top stories and display them to the user. Users can also log in and 'favorite' articles for viewing later.

## Technologies

* Angular / AngularCLI
* Firebase / Firestore
* NewsData.io API
* MaterialUI
